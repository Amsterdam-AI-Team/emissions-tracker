## Installation 

1) Clone this repository:

```bash
git clone https://github.com/Amsterdam-AI-Team/emissions_tracking.git
```




2) Install all dependencies:
    


```bash
pip install -r requirements.txt
```



The code has been tested with Python 3.11 Windows.
